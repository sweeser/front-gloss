Введение в HTML теги

HTML (HyperText Markup Language) — это основополагающий язык разметки, используемый для создания веб-страниц. Каждый элемент на веб-странице, будь то текст, изображение, ссылка или форма, представлен с помощью HTML тегов. Теги являются строительными блоками веб-документов и позволяют браузерам интерпретировать и отображать контент правильно.HTML теги имеют структурированный и иерархический формат, который помогает разработчикам логически организовывать и управлять контентом. В каждом HTML документе существует пара тегов, образующих элементы: открывающий и закрывающий теги, которые заключают в себе содержимое или другие теги. Некоторые теги также могут быть самозакрывающимися.Изучение HTML тегов является первым шагом для любого, кто хочет освоить веб-разработку. Понимание того, как правильно использовать и комбинировать теги, позволяет создавать структурированные, доступные и привлекательные веб-страницы.

Базовый HTML-код
Название тега Описание тега
<!DOCTYPE> Определяет тип документа
<html> Корневой элемент HTML-документа, содержащий всю структуру и контент страницы.
<html> Корневой элемент HTML-документа, содержащий всю структуру и контент страницы. Он определяет начало и конец HTML-документа, задавая структуру для всех остальных элементов, включая <head> и <body>. Тег <html> поддерживает несколько атрибутов, таких как lang (указание языка содержимого) и xmlns (пространство имен XML).
<head> Элемент, который содержит метаинформацию о документе, такую как заголовок, ссылки на стили и скрипты.
<title> Задает заголовок документа, который отображается в заголовке вкладки браузера.
<body> Содержит основное содержание HTML-документа, которое отображается пользователю в браузере.
<p> Определяет абзац текста. Абзацы обычно отображаются с отступом сверху и снизу, чтобы отделить их от других блоков текста.
<h1>-<h6> Задают заголовки разного уровня важности. <h1> — самый важный заголовок (обычно используется для главного заголовка страницы), а <h6> — наименее важный.
<br> Вставляет перенос строки. Этот тег самозакрывающийся и не требует закрывающего тега.
<hr> Вставляет горизонтальную линию для визуального разделения контента. Тег самозакрывающийся.
<!--....--> Определяет комментарий в HTML-коде. Комментарии не отображаются в браузере и используются для добавления пояснений или заметок в код.
Форматирование
Назване тега Описание тега
<b> Выделяет текст жирным шрифтом, не придавая дополнительного смыслового значения.
<strong> Выделяет текст жирным шрифтом, подчеркивая его важность или сильное значение.
<i> Делает текст курсивным, не придавая дополнительного смыслового значения.
<em> Делает текст курсивным, подчеркивая его акцент или выделение.
<u> Подчеркивает текст. Важно отметить, что в современных веб-стандартах подчеркивание текста обычно используется для ссылок.
<mark> Выделяет текст, как если бы он был выделен маркером.
<small> Отображает текст меньшим шрифтом, чем окружающий текст.
<del> Зачеркивает текст, указывая, что он удален или больше не актуален.
<ins> Подчеркивает текст, указывая на добавление нового контента.
<sub> Отображает текст как подстрочный (например, для химических формул или сносок).
<sup> Отображает текст как надстрочный (например, для квадратов или кубов).
<code> Форматирует текст как моноширинный, который обычно используется для обозначения фрагментов кода.
<pre> Отображает текст в виде предварительно форматированного блока, сохраняя пробелы и переносы строк. Обычно используется для отображения кода.
<blockquote> Выделяет длинные цитаты. Браузеры обычно отображают такие цитаты с отступом.
<q> Выделяет краткие цитаты в тексте. Обычно браузеры отображают такие цитаты в кавычках.
<abbr> Указывает сокращение или акроним, с возможностью добавления пояснения с помощью атрибута title.
<cite> Отображает название произведения или источника. Браузеры обычно отображают текст курсивом.
<dfn> Обозначает термин, который определяется в пределах текста.
<address> Определяет контактную информацию автора или владельца документа или статьи. Обычно отображается курсивом.
<bdi> Изолирует текст в формате bidirectional (двунаправленный текст), чтобы он отображался правильно в разных направлениях текста.
<bdo> Переопределяет текущее направление текста. Используется с атрибутом dir, который может быть равен ltr (слева направо) или rtl (справа налево).
<kbd> Определяет текст, который представляет ввод с клавиатуры, голосового ввода или любого другого ввода, отображая его моноширинным шрифтом.
<meter> Представляет скалярное значение в известном диапазоне или дробное значение, например, уровень напряженности.
<progress> Представляет индикатор выполнения задачи, показывая прогресс выполнения.
<rp> Оборачивает текст, который используется в браузерах, не поддерживающих отображение руби-аннотаций.
<rt> Определяет аннотацию или пояснение для руби-текста, обычно отображаемого над или рядом с основным текстом.
<ruby> Оборачивает руби-текст, который используется для пояснения или аннотации. Чаще всего используется в азиатских языках.
<s> Отображает текст как зачеркнутый, указывая, что он больше не актуален или удален.
<samp> Определяет образцы (sample) выводимых данных из программы или системы, отображая их моноширинным шрифтом.
<template> Определяет контейнер для HTML-контента, который не отображается при загрузке страницы, но может быть клонирован и вставлен в документ с помощью JavaScript.
<time> Определяет конкретное время или дату, которое может быть машинно читаемым. Атрибут datetime используется для указания точного времени.
<var> Определяет переменные в математических выражениях или программном коде, отображая их курсивом.
<wbr> Указывает браузеру место возможного разрыва строки, чтобы текст мог переноситься на новую строку при необходимости.
Формы и ввод
Название тега Описание тега
<form> Определяет HTML-форму для ввода данных пользователем. Включает различные элементы управления, такие как текстовые поля, кнопки и другие элементы ввода.
<input> Определяет поле ввода данных. Атрибуты type, name, value и другие используются для указания типа и параметров ввода.
<textarea> Определяет многострочное текстовое поле для ввода больших объемов текста.
<button> Определяет кнопку, которую можно нажать для выполнения действия. Атрибут type определяет, является ли кнопка отправляющей форму, сбрасывающей форму или просто кнопкой.
<select> Определяет выпадающий список. Внутри него используются теги <option> для определения доступных опций.
<option> Определяет отдельный пункт в выпадающем списке <select>.
<optgroup> Группирует несколько опций <option> внутри выпадающего списка, позволяя логически разделять элементы.
<label> Определяет текстовую метку для элемента управления формы. Атрибут for связывает метку с элементом ввода.
<fieldset> Группирует связанные элементы формы и рисует рамку вокруг них. Используется вместе с тегом <legend> для задания заголовка группы.
<legend> Определяет заголовок для содержимого внутри <fieldset>.
<datalist> Определяет список предварительно заданных опций для элемента ввода. Внутри него используются теги <option> для определения доступных опций.
<output> Отображает результат вычислений или действий пользователя, обычно используется вместе с JavaScript для динамического обновления.
